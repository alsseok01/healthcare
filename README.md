# 🏥 Healthcare - Backend ( 융합창업종합설계 )
건강 관리 어플 Pokit 의 백엔드 서버

# 🌟 핵심 기능
1. AI 기반 알약 식별 및 검색
   + 이미지 분석: 처방전이나 복약안내 봉투의 사진을 OCR을 통해 약 이름을 추출한다
   + 자연어 묘사 검색: "흰색이고 원형에 앞면에 YH 라고 적혀있어" 와 같은 묘사를 통해 해당 약에 대한 정보를 알려준다
   + 정보 보안: AI와 공공API를 모두 사용하여 정확도를 높이고 넓은 범위의 정보를 제공한다
2. 스마트 복약 알림
   + 알림 설정: 복용 시작일, 기간, 시간, 요일을 설정하여 개인화된 복약 루틴을 생성한다
   + 알림 생성 및 기록: 여러 개의 약을 복용할 경우 그룸화를 통해 관리하며, 복용 완료 여부를 기록하고 종료된 알림은 히스토리로 이관된다

# ⚙️ 실행 및 테스트 방법
application.properties에 대한 설정이 필요


Database, API keys, Google Cloud 설정


./gradlew bootRun 으로 실행
